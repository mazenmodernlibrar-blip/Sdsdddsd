
مشروع المتجر مع إضافة زر "عرض منتج" لتمكين البائعين من نشر منتجاتهم محلياً (LocalStorage).
ملفات مهمة:
- index.html : الصفحة الأساسية (محتوية على زر العرض ونموذج النشر وقسم لعرض المنتجات المنشورة)
- CSS/seller-widget.css : أنماط الواجهة الجديدة
- JS/seller-widget.js : منطق عرض/نشر المنتجات، دعم سحب الصور لليسار/اليمين، وحفظ بالـlocalStorage
كيفية الاستخدام:
- افتح index.html في المتصفح (يفضل سيرفر محلي مثل Live Server أو فتح الملف مباشرة).
- اضغط على زر 'عرض منتج' أسفل منتصف الشاشة لفتح النموذج.
- املأ الحقول وأضف صوراً (اختياري) ثم اضغط 'نشر المنتج'.
القيود:
- الصور تُحفظ كـ data URLs في localStorage (محدد إلى 8 صور لكل منتج) — هذا مناسب للتجربة المحلية. لتخزين منتجات حقيقية، يلزم رفع الصور إلى خادم/سحابة وحفظ الروابط.
- تاريخ الانتهاء يتم ضبطه تلقائياً على بعد 3 أشهر.
